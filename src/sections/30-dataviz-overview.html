Gravitacional Todo dado orbita ao redor do objeto central. O hero define o contexto — sem ele, o command center perde coerência estrutural.
Hierárquico Topo = macro (KPIs). Meio = análise (charts). Base = detalhe (tabelas). A camada mais alta tem menos informação e mais peso visual.
Vivo Dados em tempo real. O command center nunca é estático — cada atualização é um sinal, não um ruído. Motion funcional comunica mudança.
